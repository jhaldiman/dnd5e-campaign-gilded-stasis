## Handout 7 — Valerius's Personal Journal (Fragment)

*Found in the Citadel of Chronos — Mirror Gallery or Anchor Chamber.*

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  Year 14, Day 203                                    │
│                                                      │
│  The conjunction approaches. Fourteen years of        │
│  work. Fourteen years of smiling at banquets,        │
│  shaking hands, pretending that patience is the      │
│  same as peace.                                      │
│                                                      │
│  I watched a child die of fever last week, in the    │
│  lower quarter. His mother held him and asked me     │
│  why. I had no answer. But I will.                   │
│                                                      │
│  Soon, no child will die of fever. No mother will    │
│  hold a cooling body. No plague, no famine, no war.  │
│  Forever Noon. A world at its best, held there by    │
│  my hand, because no one else had the courage.       │
│                                                      │
│  They call it tyranny. I call it mercy.              │
│                                                      │
│  The heroes in the Hall — they gave more than they   │
│  know. Their strength will fuel a world without      │
│  suffering. Without change. Without that terrible,    │
│  grinding entropy that makes monsters of us all.     │
│                                                      │
│  Elara asked me today if I ever doubted. I told      │
│  her no.                                             │
│                                                      │
│  I lied.                                             │
│                                                      │
│  I doubt every day. And every day I watch the        │
│  world suffer, and the doubt goes away.              │
│                                                      │
│                                    — V.T.            │
│                                                      │
└──────────────────────────────────────────────────────┘
```

**DM Note:** This is the most sympathetic Valerius will ever be on paper. If a
player reads it aloud, let the table sit with it. The best villains make you
*understand*.

## Handout 9 — Elara's Letter (Epilogue)

*Give to the players after the final battle, if Elara survived.*

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  I have written this letter eleven times. Each       │
│  version was longer. None of them said what I        │
│  needed to say. So:                                  │
│                                                      │
│  I was wrong.                                        │
│                                                      │
│  I followed a man I believed was saving the world,   │
│  and I helped him cage it. I looked at you — my      │
│  friends, the people I was supposed to protect —     │
│  and I called you traitors, because admitting the    │
│  truth would have meant admitting that my entire     │
│  life was a lie.                                     │

│                                                      │

│                                                      │

│  It doesn't look like freezing the world in amber.   │

│                                                      │
```
